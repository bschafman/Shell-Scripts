Remove All iMessage Logs

A shell script designed to remove any and all iMessage logs on MacOS. For both iMessage native MacOS logs, as well as transferred logs from any iOS device.

Here is the direct link to the GitHub code repo for this script

iMessage log files are annoying on MacOS. There is no simple way to disable them by default in the application. So instead, this script will let you rest easy knowing your boss will not find your work-hours chat log to your girlfriend (or drug dealer.. if thats your thing. heh)

This script is stupid simple.. But makes it easy to delete chat logs on your (or your employers) apple computer. I found this task tedious and repetitive and not as easy as it should be.

To use:

Save the file to your hard drive as simply "remove_chat_logs" (no extension). It is already an executable file, so just double click it and follow the instructions.

Now, just double click the file.

Note- no files will ever be deleted without you first giving the OK.

Keep the file on the desktop (or wherever you want to put it) and run it as often as you want.

Email me at joshpopenyc@gmail.com if you have questions. Or if you have hatemail. I'm always up for a good read.

NOTE: Files are NOT recoverable after deletion. This will not send them to the trash- they are gone forever. This is why it prompts you (y/n) to continue with deletion when files are found.
